El Juego

¿Qué es el Secret Oracle?

El Secret Oracle es un juego conversacional donde una IA guarda una imagen en secreto. Solo tú recibes las pistas — descripciones crípticas y poéticas generadas por modelos de lenguaje avanzados — y debes descifrar qué imagen está ocultando el Oráculo.

Cada partida es única. Las imágenes son generadas por IA, las pistas son únicas para cada imagen, y la visión computacional valida tu respuesta. Un backend serverless en AWS orquesta todo en tiempo real.

Puedes pedir más pistas, enviar múltiples intentos o rendirte para revelar el secreto. Cuando aciertas — o te rindes — el Oráculo descorre el velo y revela la imagen oculta.

01

Nueva partida

El backend genera una imagen con IA y retorna la primera pista críptica al jugador.

02

Enviar intento

El jugador escribe su hipótesis en el chat. El backend decide si es correcto o no y responde con retroalimentación.

03

Pedir pista extra

Si el jugador se queda sin ideas, puede solicitar nuevas pistas. Hay un número limitado disponibles.

04

Revelar o rendirse

Al acertar o rendirse, el backend devuelve la imagen real y las etiquetas detectadas por visión computacional.

Infraestructura

Construido sobre AWS + IA

Cada componente del Secret Oracle vive en AWS. Aprenderás a conectar servicios de cómputo, almacenamiento, APIs y modelos de inteligencia artificial en una arquitectura serverless real.

⚡
AWS Lambda
Lógica de negocio serverless. Cada operación del juego — iniciar, adivinar, revelar — es una función Lambda independiente.
Compute
🌐
API Gateway
Expone los endpoints REST del juego. Maneja rutas, autenticación y CORS de forma declarativa.
Networking
🧠
Amazon Bedrock
Modelos fundacionales para generar imágenes con IA y producir las pistas crípticas que guían al jugador.
Generative AI
👁️
Amazon Rekognition
Visión computacional que analiza la imagen secreta y extrae etiquetas para validar los intentos del jugador.
Computer Vision
🗄️
Amazon DynamoDB
Base de datos NoSQL para persistir sesiones, intentos, pistas y el estado de cada partida en tiempo real.
Storage
🚀
AWS SAM / CDK
Infraestructura como código. Desplegarás toda la arquitectura con un solo comando usando SAM o CDK en TypeScript.
IaC

▸ Flujo de una partida en tiempo real

Frontend → API Gateway → Lambda → Bedrock + Rekognition + DynamoDB
